# AX 안티패턴: 흔한 실수와 개선법

사업관리본부 직원들이 Claude Code를 활용할 때 자주 하는 실수 패턴입니다.

---

## 공통 안티패턴

### 1. 빈 요청 (Empty Request)

**패턴:** 배경 없이 "해줘"만 말하기

❌ "보고서 써줘"
❌ "번역해줘"
❌ "이거 정리해줘"

✅ "나는 사업관리본부 예산 담당이야. 3월 예산 집행 현황 데이터를 첨부했어. 팀장 보고용 요약 보고서를 작성해줘. A4 1페이지 이내로."

**왜 문제인가:** AI가 맥락 없이 일반적인 결과를 만들어냄. 결과를 받아도 수정이 많이 필요함.

---

### 2. 무검증 수용 (Rubber Stamping)

**패턴:** AI 결과를 검토 없이 그대로 사용

❌ AI가 작성한 보고서를 바로 제출
❌ AI가 계산한 숫자를 확인 없이 사용
❌ AI가 만든 이메일을 수정 없이 발송

✅ "이 보고서에서 숫자가 맞는지 확인해줘"
✅ "경영진 관점에서 이 내용이 적절한지 검토해줘"
✅ "이 문장이 공식 문서에 어울리는지 봐줘"

**왜 문제인가:** AI는 틀릴 수 있음. 특히 숫자, 날짜, 이름, 맥락 오류가 많음.

---

### 3. 주방 싱크대 요청 (Kitchen Sink)

**패턴:** 한 번에 너무 많은 것을 요청

❌ "보고서 작성하고, 데이터 분석하고, 발표 자료 만들고, 이메일도 써줘"

✅ 단계별로 나눠서 진행:
1. "먼저 데이터 분석 결과 요약해줘"
2. "분석 결과를 바탕으로 보고서 구조 잡아줘"
3. "구조 확정 후 각 섹션 내용 채워줘"

**왜 문제인가:** AI가 모든 것을 한꺼번에 처리하면 각 부분의 품질이 떨어짐.

---

### 4. 오류 무시 (Error Blindness)

**패턴:** AI가 오류나 한계를 말해도 무시하고 계속 진행

❌ AI: "이 데이터만으로는 정확한 분석이 어렵습니다" → 그냥 "계속해줘"

✅ AI가 한계를 말하면 → "그러면 어떤 데이터가 있으면 가능해?", "대안은?"

**왜 문제인가:** 불완전한 정보 기반 결과물이 나옴. 나중에 더 큰 수정이 필요.

---

### 5. 형식 미지정 (Format Blindness)

**패턴:** 출력 형식을 지정하지 않음

❌ "이거 분석해줘" → 긴 문단으로 나옴 (표가 필요했는데)

✅ "분석 결과를 표로 만들어줘. 컬럼: 항목명, 금액, 전월 대비 변화율"

**왜 문제인가:** 원하는 형식과 다른 결과가 나와 재작업 필요.

---

## 역할별 안티패턴

### 문서 작성자

**독자 미지정:** "보고서 써줘" → 누가 읽는지 말 안 함
→ ✅ "팀장 보고용", "경영진 보고용", "외부 제출용" 중 명시

**길이 미지정:** 형식 없이 작성 요청 → 너무 길거나 짧은 결과
→ ✅ "A4 1페이지", "슬라이드 5장", "이메일 3~5줄" 지정

---

### 데이터 분석자

**데이터 미첨부:** "이 데이터 분석해줘" → 데이터를 안 붙여넣음
→ ✅ 데이터를 먼저 붙여넣고 분석 요청

**맥락 없는 숫자:** "이 숫자들 정리해줘" → 단위, 기준, 기간 없음
→ ✅ "2026년 3월, 원 단위, 전월 대비 비교"

---

### 소통 조율자

**톤 미지정:** "이메일 써줘" → 너무 격식적이거나 격식 없음
→ ✅ "공식적인 톤으로", "친근한 톤으로", "대외 협력사 대상으로"

**참석자 미지정:** "회의록 작성해줘" → 담당자 없는 회의록
→ ✅ 참석자 목록 + 각자 역할 함께 제공

---

### 업무 자동화자

**프로세스 설명 부족:** "이 작업 자동화해줘" → 현재 과정 설명 없음
→ ✅ 현재 과정을 단계별로 설명 후 자동화 요청

**제약 조건 미제시:** 회사 시스템, 권한, 도구 제한을 말 안 함
→ ✅ "우리 회사는 [시스템]을 씀, [도구]는 사용 불가"

---

## 개선의 신호 (잘하고 있는 것들)

✅ "배경: ... / 요청: ... / 형식: ..." 구조로 시작
✅ AI 결과에 "이 부분이 맞나요?", "다른 방법은?" 질문
✅ 긴 작업을 단계로 나눠 진행
✅ "이 방법의 한계가 뭐야?", "대안은?" 물어봄
✅ AI 결과에 본인의 전문 지식을 추가해서 완성
