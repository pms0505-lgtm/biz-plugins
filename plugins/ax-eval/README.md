# AX-Eval: 내 AI 활용 수준 자동 측정 플러그인

Claude Code를 얼마나 잘 쓰고 있는지 자동으로 분석해드립니다.
별점 5단계 레벨 + 약한 부분 맞춤 팁까지.

---

## 설치 전 확인사항

| 항목 | 확인 방법 | 필요 버전 |
|------|-----------|-----------|
| Claude Code | `claude --version` | 최신 버전 |
| Python | `python --version` | 3.8 이상 |

> Python이 없으면 IT 담당자에게 문의하세요.

---

## 설치 절차

### 1단계 — 플러그인 설치

Claude Code 터미널에서 아래 명령어를 입력합니다.

```
claude plugins install https://github.com/DG-BusinessMgmt/ax-eval
```

설치가 완료되면 아래 메시지가 나옵니다:

```
✓ ax-eval 플러그인이 설치되었습니다.
```

---

### 2단계 — 초기 설정 (딱 한 번만)

Claude Code를 열고 입력합니다:

```
/ax-eval 시작
```

그러면 Claude가 아래처럼 안내합니다:

```
안녕하세요! AX-Eval입니다.

이 시스템은 여러분의 Claude Code 활용 수준을 분석하여
성장을 추적하고 맞춤 팁을 드립니다.

설정은 딱 2분이면 끝납니다. 시작할게요!

어떤 업무를 주로 하시나요?

1. 문서 작성 — 보고서, 기획서, 제안서 작성
2. 데이터 분석 — 데이터 정리, 분석, 시각화
3. 소통 조율 — 회의록, 이메일, 커뮤니케이션
4. 업무 자동화 — 반복 업무 효율화, 프로세스 개선
```

번호를 입력하면 설정이 완료됩니다.

> **주의**: Claude Code를 최소 3번 이상 사용한 후 설치해야 정확한 분석이 됩니다.

---

### 3단계 — 첫 번째 레벨 체크

설정 완료 직후 바로 확인해보세요:

```
/ax-eval 체크
```


## 이후 사용법

| 명령어 | 언제 쓰나요 | 주기 |
|--------|------------|------|
| `/ax-eval 체크` | 내 레벨 확인 | 주 1회 권장 |
| `/ax-eval 팁` | 어떻게 성장하지? | 필요할 때 |
| `/ax-eval 시작` | 처음 설치 시 | 1회만 |

---

## 자동 피드백 기능

설치 후에는 **아무것도 안 해도** 됩니다.

매주 Claude Code를 열면, Claude가 자동으로 약한 축을 인식하고 대화 중에 자연스럽게 알려줍니다.

```
김지수: 3월 예산 집행 현황 표 만들어줘

Claude: (표 작성 후...)
        혹시 합계나 항목별 집행률이 맞는지 한 번 더 확인해드릴까요?
        숫자 검증을 같이 하면 보고서 신뢰도가 올라가거든요.
```

> 직접 피드백을 보고 싶으면 `/ax-eval 체크`를 실행하세요.

---

## 평가 체계

### 5단계 레벨

| 레벨 | 이름 | 어떤 상태인가요 |
|------|------|----------------|
| ⭐ | 입문 | "해줘"만 말하는 단계 |
| ⭐⭐ | 활용 | 구체적으로 시키기 시작 |
| ⭐⭐⭐ | 협업 | AI와 대화하며 결과를 다듬음 |
| ⭐⭐⭐⭐ | 주도 | 업무를 쪼개서 AI에게 맡김 |
| ⭐⭐⭐⭐⭐ | 전략 | AI를 업무 전략에 녹여냄 |

### 4가지 측정 축

| 축 | 측정하는 것 |
|----|------------|
| **요청력** | AI에게 배경·목적·형식을 얼마나 잘 전달하는가 |
| **검증력** | AI 결과를 그냥 쓰지 않고 확인하고 수정하는가 |
| **활용력** | Claude Code의 다양한 기능을 쓰는가 |
| **판단력** | AI를 언제, 어떻게 쓸지 전략적으로 판단하는가 |

## 자주 묻는 질문

**Q. 설치하면 제 대화 내용이 외부로 나가나요?**
아니요. 분석은 내 컴퓨터 안에서만 이루어집니다. `~/.claude/` 폴더의 로그만 읽으며, 외부 서버로 전송되지 않습니다.

**Q. 레벨이 너무 낮게 나왔어요.**
처음엔 낮게 나오는 게 정상입니다. Claude Code를 2주 정도 꾸준히 쓰면서 `/ax-eval 체크`로 변화를 확인해보세요.

**Q. 분석할 세션이 없다고 나와요.**
Claude Code를 3번 이상 대화한 후 `/ax-eval 체크`를 다시 실행해보세요.

**Q. 역할을 잘못 선택했어요.**
`/ax-eval 시작`을 다시 실행하면 역할을 재선택할 수 있습니다.
