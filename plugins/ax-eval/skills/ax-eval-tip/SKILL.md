# ax-eval-tip: 맞춤 활용 팁

현재 AX 레벨에 맞는 Claude Code 활용법과 프롬프트 템플릿을 제공합니다.

---

## 실행 순서

### 0단계: 최신 Anthropic 팁 수집

다음 쿼리로 웹서치하여 현재 레벨에 맞는 Claude/Claude Code 최신 활용법을 수집합니다:

```
"site:docs.anthropic.com claude prompt engineering best practices"
"anthropic claude code tips features 2025"
```

검색 결과에서 실용적인 팁 1~2개를 추출하여 메모합니다.
검색 실패 시 이 단계를 건너뜁니다.

### 1단계: 현재 레벨 파악

`~/ax-eval/assessments/` 에서 가장 최근 평가 파일을 읽습니다.

파일이 없으면: "아직 레벨 평가가 없습니다. 먼저 `/ax-eval 체크`를 실행해주세요."

읽어야 할 정보:
- `level` (1~5)
- `role` (역할)
- `scores.요청력`, `scores.검증력`, `scores.활용력`, `scores.판단력`

### 2단계: 가장 약한 축 파악

4축 점수 중 가장 낮은 축을 식별합니다. 이 축에 맞는 팁을 우선 제공합니다.

### 3단계: 레벨별 콘텐츠 출력

아래 레벨별 팁 섹션에서 현재 레벨에 맞는 내용을 출력합니다.

---

## 레벨별 팁 콘텐츠

### ⭐ 입문 단계

**이번 주 추천 습관:**
AI에게 요청할 때 배경을 한 줄 먼저 설명해보세요.

**기본 프롬프트 구조:**
```
나는 [부서/역할]에서 [업무]를 하고 있어.
[배경 설명을 1~2줄].
[구체적으로 해줬으면 하는 것].
```

**예시 (Before → After):**

❌ "보고서 써줘"

✅ "나는 사업관리본부에서 예산 관리를 담당하고 있어.
3월 예산 집행 현황을 정리해야 해.
항목별 집행률과 이슈 사항을 표로 만들어줘."

**이번 주 목표:** 1가지 업무에 AI 적용해보기

---

### ⭐⭐ 활용 단계

**이번 주 추천 습관:**
AI 결과를 받은 후 "이거 맞아?" 하고 한 번 더 확인해보세요.

**활용 프롬프트 구조:**
```
나는 [부서]에서 [업무]를 담당해.
[배경: 상황, 데이터, 조건].
[요청: 구체적으로 무엇을].
[형식: 표/목록/문단 중 어떤 형태로].
```

**검증 습관 예시:**
```
(AI 결과를 받은 후)
"방금 만들어준 표에서 합계가 맞는지 확인해줘."
"다른 방식으로 표현하면 어떻게 될까?"
"경영진이 보기에 이 보고서가 충분한지 검토해줘."
```

**이번 주 목표:** AI 결과를 최소 1번 검토 후 수정 요청

---

### ⭐⭐⭐ 협업 단계

**이번 주 추천 습관:**
복잡한 업무를 단계로 나눠서 AI에게 맡겨보세요.

**업무 분해 프롬프트:**
```
[업무명]을 단계별로 나눠줘.
그리고 각 단계에서 내가 직접 해야 할 것과 AI가 도울 수 있는 것을 구분해줘.
```

**역할별 프롬프트 템플릿:**

📄 **문서 작성:**
```
보고서 구조 먼저 잡아줘:
- 목적: [보고서 목적]
- 대상 독자: [누가 읽는지]
- 핵심 메시지: [전달할 핵심 1가지]
- 포함할 섹션: [개요/현황/분석/결론]
구조가 OK면 각 섹션 내용을 순서대로 작성할게.
```

📊 **데이터 분석:**
```
다음 데이터를 분석해줘: [데이터 붙여넣기]
1. 주요 트렌드 3가지
2. 이상치나 특이사항
3. 다음 달 예측 (가능하면)
표와 요약 문장으로 정리해줘.
```

📧 **소통 조율:**
```
다음 회의 내용으로 회의록을 작성해줘: [회의 내용]
형식:
- 참석자:
- 주요 논의 사항 (항목별)
- 결정 사항
- 다음 할 일 (담당자, 기한)
```

🔄 **업무 자동화:**
```
매주 반복하는 [업무명] 작업이 있어.
현재 과정: [1. 단계1, 2. 단계2, 3. 단계3]
이 중에서 AI가 대신할 수 있는 단계는 어떤 건지,
어떻게 자동화할 수 있는지 알려줘.
```

**이번 주 목표:** 1가지 업무를 단계별로 나눠서 진행

---

### ⭐⭐⭐⭐ 주도 단계

**이번 주 추천 습관:**
AI 답변을 그대로 쓰지 말고, 내 전문성을 더해 완성하세요.

**고급 프롬프트 패턴:**

**제약 조건 명시:**
```
[요청] 해줘.
제약 조건:
- 분량: A4 1페이지 이내
- 대상: [이해관계자]
- 사용하면 안 되는 표현: [금지어]
- 반드시 포함할 내용: [필수 항목]
```

**대안 비교 요청:**
```
[문제 상황]에 대해 3가지 접근 방법을 비교해줘.
각각의 장단점과 우리 상황에 맞는 추천을 포함해서.
```

**비판적 검토:**
```
내가 [기획/보고서/계획]을 작성했어: [내용]
이것의 약점이나 빠진 부분을 지적해줘.
특히 [이해관계자]가 반대할 수 있는 포인트 위주로.
```

**이번 주 목표:** AI 결과의 80%를 활용하고, 20%는 내 판단으로 보완

---

### ⭐⭐⭐⭐⭐ 전략 단계

**이번 주 추천 습관:**
AI 활용 노하우를 팀원과 공유하는 문서를 만들어보세요.

**조직 레벨 활용 패턴:**

**팀 가이드 작성:**
```
우리 팀에서 [업무 유형]에 AI를 활용하는 표준 절차를 만들어줘.
신입 팀원도 바로 쓸 수 있도록 구체적인 프롬프트 예시 포함.
```

**업무 개선 제안:**
```
사업관리본부의 [현재 프로세스]를 AI로 개선하면 어떤 효과가 있을지 분석해줘.
예상 시간 절감, 품질 향상, 리스크 포인트를 포함한 도입 제안서 형태로.
```

**이번 주 목표:** 팀원 1명에게 AI 활용법 전수하기
